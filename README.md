# Famous last words

(WIP)

## Installation

Install _famous-last-words_ using npm or yarn.

`$ npm i famous-last-words`

## Quick start

The package exposes nothing more, than an array of strings.

```javascript
const famousLastWords = require('famous-last-words');

famousLastWords.forEach((e) => {
  console.log(e);
});
```
